Automated Turret

Embedded · Computer Vision · Control Systems

An autonomous targeting system that uses a YOLO object detection model to identify and track moving targets in real time, commanding a pan-tilt mechanism via a PID controller to lock on and engage with high accuracy.

  • Deployed YOLOv8 for real-time color and object detection with live tracking feedback loop
  • PID controller handles pan/tilt under heavy payload; mitigates ESC motor and solenoid electrical noise
  • IMU + complementary filter fuses accelerometer and gyroscope data to isolate drift from vibration
  • Conducted full prototyping cycle and Critical Design Review (CDR) with technical peer panel
~2s Target lock-on
95% Hit accuracy
5°/s Tracking speed
